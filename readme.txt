Integrate DCash with WooCommerce.

== Description ==

This plugin creates a new WooCommerce payment gateway that allows your store to take DCash payments. Customers place orders and checkout using their DCash wallet. Once the payment is complete, the funds arrive on your DCash Merchant wallet and the order is completed.

Uriahs Victor/SoaringLeads is in no way affiliated with DCash or the ECCB. This plugin was developed on top of the DCash API with assistance from the ECCB in providing revelant documentation. 

Need more information about DCash? Find out more [here >>>](https://www.dcashec.com/)

== Installation ==
1. Upload "integrate-dcash-with-woocommerce" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==
= Where do I find my API Key? =
Your API key can be found in the "E-Commerce API Key" area of your DCash Merchant App.

= What is sandbox mode? =
Sandbox mode allows you to test payments without needing real DCash on your account. To use Sandbox mode you need access to the DCash Merchant Beta App from ECCB.
